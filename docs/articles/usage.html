Usage

Each codec extends BinaryCodec, which provides the implementation that is common to all codecs. At it simplest, each codec provides GetBytes(String), GetChars(Byte[]) and similar overloads. The methods decode and encode sequences of characters and bytes that are completely loaded into memory.

As an example, encoding an array of bytes with the Base16 codec can be achieved with:

var inputData = new byte[]{0, 1, 2, 3, 4, 5};
var encodedData = BinaryCodecs.Base16Standard.GetChars(inputData);
var decodedData = BinaryCodecs.Base16Standard.GetBytes(encodedData);

Streaming

When you have an indeterminate amount of input or your simply don't want to load all of the data into memory, use the BinaryDecoderStream or BinaryEncoderStream classes.
